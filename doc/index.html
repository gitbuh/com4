Introduction

CoM4 is a build-time implementation of CommonJS Modules 1.1.1 built on the M4 macro preprocessor.

Download

Download the latest version of CoM4 here.

Installation

Download the archive, extract it to a location of your choice, set the executable flag on the file named "com4", and create a symbolic link to it in your path. For example: 


cd ~
tar -zxf com4.tgz
chmod +x com4/com4
sudo ln -s ~/com4/com4 /usr/bin/com4

Usage


com4 -i <infile.js> -o <outfile.js>
